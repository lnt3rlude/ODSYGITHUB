# 📝 Застосунок керування записами (ЛР1)

Цей проєкт — це односторінковий веб-застосунок (SPA) для створення, редагування, видалення та фільтрації записів користувачів. Проєкт реалізовано з використанням базових веб-технологій без використання додаткових бібліотек чи фреймворків (Vanilla JS). Має сучасний темний інтерфейс (Dark Mode) з ефектами скла (Glassmorphism) у стилі macOS.

## ✨ Основний функціонал (CRUD)

- **Створення записів (Create):** Додавання нових користувачів із зазначенням імені (UserName), дати, типу доступу (Read/Write/Admin), коментарів та статусу.
- **Перегляд (Read):** Відображення всіх збережених записів у вигляді зручної адаптивної таблиці.
- **Редагування (Update):** Швидке завантаження даних існуючого запису назад у форму для їх зміни.
- **Видалення (Delete):** Безпечне видалення записів із попереднім підтвердженням дії.

## 🔍 Додаткові можливості

- **Пошук та фільтрація:** - Динамічний пошук (в реальному часі) за полями `UserName` або `Comments`.
  - Зручна фільтрація таблиці за статусом (`Pending`, `Approved`, `Rejected`).
- **Збереження даних:** Усі записи автоматично зберігаються у пам'яті браузера користувача за допомогою `localStorage`. Дані не зникають після оновлення сторінки чи закриття вкладки.
- **Валідація форми:** Інтерактивна перевірка правильності заповнення полів (обов'язкові поля, обмеження на довжину тексту) з підсвічуванням помилок.

## 🛠 Використані технології

- **HTML5:** Семантична розмітка структури застосунку.
- **CSS3:** Сучасна стилізація (Flexbox, градієнти, `backdrop-filter` для ефекту матового скла, CSS-транзиції).
- **JavaScript (ES6+):** Логіка роботи застосунку, маніпуляції з DOM, робота з Local Storage, обробка подій.

## 🚀 Як запустити проєкт

Оскільки проєкт є повністю клієнтським застосунком, для його запуску не потрібні сервери чи бази даних.
